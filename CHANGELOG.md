## [1.0.0] - 2026-03-27

### Added

- Initial release of ExecAI
- `executive_ai.py` : full executive AI pipeline in a single file (plan, execute, report)
- `test_executive_ai.py` : unit tests with mocking for all three core functions
- GitHub Actions CI workflow (`ci.yml`) : runs tests on every push and pull request
- Issue templates for bug reports and feature requests
- Pull request template
- `scripts/setup.sh` : one-command installation script
- `docs/architecture.md` : detailed technical architecture documentation
- Full project documentation (README, CONTRIBUTING, SECURITY, CODE_OF_CONDUCT)
