# Code of Conduct

## Our Standards

This project is open to everyone. We expect all participants to behave in a way that makes the community welcoming and productive.

Acceptable behavior includes :

- Using clear, respectful language
- Accepting constructive feedback without defensiveness
- Focusing on what is best for the project and its learners
- Helping newcomers understand the code and concepts

Unacceptable behavior includes :

- Insults, personal attacks, or condescending language
- Harassment of any kind, public or private
- Dismissing beginner questions or contributions

---

## Enforcement

Violations can be reported to convergence-tech@proton.me.

Reports will be reviewed confidentially. Confirmed violations may result in temporary or permanent removal from the project's spaces.

---

## Scope

This Code of Conduct applies in all project spaces : GitHub issues, pull requests, discussions, and any other channel associated with this project.
